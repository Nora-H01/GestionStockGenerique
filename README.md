# Systčme de Gestion de Stocks Générique en C#

## Description

Ce projet est un exercice de programmation orientée objet en C#.  
L'objectif est de créer un systčme de gestion de stocks pour un magasin pouvant contenir différents types de produits (livres, électronique, vętements, etc.).

Le projet illustre l'utilisation de :  
- Interfaces génériques  
- Classes génériques  
- Contraintes de type  
- Méthodes génériques  
- Programmation orientée objet en C#

## Technologies

- C#  
- .NET (Console Application)  
- Programmation orientée objet  
- Collections génériques
